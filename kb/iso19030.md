# ISO 19030 船體與螺槳效能量測方法（摘要）

## Speed Loss 定義

ISO 19030 將 speed loss 定義為：相同軸功率下，預期船速與實測船速的百分比差。
預期船速來自「參考期」（通常為塢修或清洗後）建立的速度–功率基準曲線。
Speed Loss 是相對於參考期的值，非相對理想乾淨船體的絕對值。

## 三層方法

- 第一層：專用高精度感測（軸功率計、對水速度計 STW）。
- 第二層：既有自動記錄資料。
- 第三層：正午報表（最低資料需求層）——本系統所在層級。以油耗作為功率代理，
  航速多為對地速度 SOG，洋流影響為主要不確定性來源。

## 四個 KPI

1. 塢修效果（dry-docking performance）
2. 在航效能（in-service performance）
3. 維護觸發（maintenance trigger）——本系統的清洗建議對應此 KPI
4. 維護成效（maintenance effect）——清洗前後 speed loss 落差

## 資料篩選慣例

排除極端天候（如 Beaufort 4 級以上）與非穩態航行，僅保留全速航行時段，
以降低環境雜訊對基準比較的污染。
